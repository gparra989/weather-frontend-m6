# 🌦️ weather-frontend-m6

Aplicación de clima desarrollada en **HTML, CSS, Javascript y Vue** que permite consultar (mediante plantillas) el estado actual 
y el pronóstico del tiempo en diferentes ciudades, usa una base de datos de ejemplo usando json server desde donde se obtienen los datos. 

La sección **Estadísticas de la Semana** busca la temperatura miníma y máxima de la semana, así como el promedio y el estado general.

Esta aplicación hace uso de Vue Router y Json Server. 

Contamos con 5 componentes, 2 principales: HomeComponent y DetalleComponent, y 3 componentes para darle más estilos a la página: AcercaDeComponent, NavbarComponent y FooterComponent. 

---

## 🚀 Características
- Visualización de temperatura, humedad, velocidad del viento y condiciones generales.
- Pronóstico semanal.
- Interfaz responsiva y amigable.
- Se utiliza la metodología BEM.
- Se emplea Sass para el uso de los colores, mixins, y parciales principales como card, hero, general, los cuales agrupan la base de la página web asi como los componentes visuales. 

---

## 🛠️ Tecnologías utilizadas
- **Frontend:** HTML, CSS, BEM, Sass, JavaScript.
- **Extras:** Bootstrap 5.
